Upload

1. Upload a file

2. View its size

API

POST /post

A simple API that returns the size of an uploaded file.

Request

Send a file. Use multipart POST. 

$ curl -F "blah=@/tmp/file.bin" http://localhost:3100/post
Response

The JSON payload of the response includes keys for fileName and fileSize in bytes. 

{"fileName":"file.bin","fileSize":59462}
